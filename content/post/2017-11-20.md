> **** (@; 24/5): #rstats shiny tip of the day:
>
Need to generate *several* UI elements inside a renderUI()? Use tagList() to group multiple UI elements together.
>
Example:
output$x &lt;- renderUI({
  tagList(
    textInput(...),
    numericInput(...)
  )
